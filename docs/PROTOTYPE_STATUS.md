# 原型状态与路线图

## 当前已实现：猫咪城镇·第一日

- 船上第一人称 3D：WASD/方向键移动、鼠标观察、F 互动；窗口、打包台、贴标台、仓储格位和下岛入口均使用基础几何体。
- 首日 7 条固定订单：阿岚的教学订单，以及露娜、诺亚、贝洛、悠悠、阿棉、芸姐相关的城镇日常订单。
- 每条订单保留独立的窗口、物品实体、包装状态、面单、特殊标识、保存条件、仓储格位、报酬与声望值。
- 完整白天流程：主动接待客户 → 在同一窗口取货 → 打包生成地址面单 → 特殊件贴标 → 玩家自由存入正确的具体格位；可取出纸箱重新整理。
- 夜晚确认弹窗、2 格手动背包、玩家主动选择仓储格位装载。
- 猫咪城镇横版配送：图书室、裁缝店、钟楼、小剧团、港口诊所、潮汐壶茶馆各有独立投递范围；投错地点不会消耗包裹。
- 底部 NPC 对话框：姓名、颜色头像占位、订单说明、投递反馈和后续剧情线索；阿岚完成继承邮船的开场教学。
- 金币与猫咪城镇声望结算；第一日清空订单后显示日结，并进入下一日。

## 手动验收建议

1. 启动项目，使用空格或回车读完阿岚开场对话。
2. 在任意窗口依次按 F：阅读订单、接单、再次按 F 拿起物品。
3. 到打包台封箱；阿岚订单须拿起纸箱并到贴标台完成“易碎”标识，再放进常温货架。
4. 重复接收其他订单，确认鲜花只能放温室、冷藏药剂只能放冷冻室。
5. 到下岛入口确认夜晚，从指定格位装入 1–2 件包裹，再下岛。
6. 依据箱子面单到对应店铺；错误店铺不会投递，正确店铺出现不同 NPC 对话与金币/声望反馈。
