# 当前美术与叙事方向

## 已确认的边界

- 当前阶段优先制作环境、设备、包裹与岛屿；暂不设计玩家主角。
- 邮政系统目前是日常、可信的海上服务；魔法、奇异航线等设定以后再考虑。
- 所有居民都是猫咪世界中的拟人动物，空间尺度与人类一致。

## 氛围

整体基调是温暖日常：一个随时可以回船、喝杯热茶、整理包裹的世界。

部分故事可以有轻微忧伤，例如坚守灯塔的老人、失联已久的朋友或迟到的信；但画面不能走向阴森、灾难或绝望，而应保留温暖灯光、被使用的生活物件与“关系仍有机会修复”的感觉。

## 环境关键词

- 圆润低模、略带手绘笔触的材质；
- 厚实木梁、木地板、金属货架、模块化设备；
- 海上黄昏的冷蓝环境光，对比船舱与窗户透出的暖黄灯光；
- 丰富但不阻塞动线的包裹、航线图、植物、茶杯、雨衣和工具；
- 有人在认真生活与工作，而不是空旷的样板间。

## 近期环境优先级

1. 船舱邮局整体空间。
2. 接件柜台、打包台、标签机、常温货架、冷冻室、温室。
3. 邮船外观和码头。
4. 北礁灯塔岛。
5. 热闹沙滩、森林驻地、猫咪城镇。
