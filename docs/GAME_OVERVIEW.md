# 游戏整体玩法

## 一句话概念

《月光邮船》是一款温暖的海上邮政生活游戏。玩家继承爷爷留下的邮船，在白天处理居民主动寄来的物品，在夜晚选择包裹登陆岛屿完成投递；通过一次次可靠的送达，重新连接群岛居民，也逐步把船与岛上的住处变成属于自己的家。

## 世界观与开场

玩家的爷爷曾是这片群岛的邮船船长。爷爷去世后，玩家接手这艘船与尚未中断的航线。

开场时，一位爷爷的老顾客来到接待厅窗口寄件。玩家告知对方爷爷去世的消息；这位老顾客既有自己的生活与后续故事，也会在第一天以自然的方式引导玩家完成收件、打包、入库和夜间投递。

爷爷的过去会作为一条长期暗线存在：旧航线图、被划掉的停靠点、未寄出的信、账本记录，以及居民偶尔提起的往事。前期不将其做成悬疑主线；玩家先建立自己的日常，之后才逐渐理解爷爷曾维系的联系。

## 基调

世界由拟人猫咪居民构成，是略带童话感、但可信的海岛日常。整体基调温暖、安静、有生活感；少数角色故事可以有轻微忧伤，但不走灾难、阴森或绝望路线。

邮船不是热闹社交场所，重点仍是可靠的寄送、工作空间与玩家和居民之间逐渐建立的信任。

## 一天的核心循环

```text
白天：船上邮局
玩家主动到窗口接单
→ 从实际接件窗口拿取物品
→ 带到打包台封箱，并完成收件人、地址与保存条件面单
→ 有特殊要求时，在贴标台添加易碎、冷藏急件等标识
→ 将纸箱手动放入自己选择的正确仓储格位
→ 可随时拿起已入库包裹重新整理

夜晚：海岛配送
玩家从具体仓储格位主动挑选包裹
→ 装入有限容量的配送背包
→ 下岛探索
→ 选择对应包裹并投递
→ 自行回到邮船
→ 清空当夜配送后开始新的一天
```

白天强调手作、空间和整理；夜晚强调选择、探索和人物关系。没有排队或强倒计时压力，客户只会在玩家主动准备接单时出现。

## 包裹与仓储

每个订单包含：寄件人、物品、收件人、收件地址、包装类型、保存条件、可选特殊标识、运费，以及可选剧情标记。

打包后的纸箱会显示实体面单，帮助玩家在货架上识别目的地、收件人、保存条件和特殊要求。仓库由多个真实格位构成：常温、冷冻和温室区域各有容量与保存限制。整理方式由玩家决定；夜晚带哪几件包裹也由玩家自己选择。

大部分订单是支撑日常经营的普通快递。少数特殊订单会推进 NPC 关系、解锁配方、设施或地点声望。

## 群岛地点

- **北礁灯塔**：安静的海风、灯塔维护物资、旧信与守望人的故事。
- **沙滩景点**：餐馆冷链、游客物品、节庆与热闹的商业订单。
- **森林探险队驻地**：补给、种子、样本、地图和队员之间的关系变化。
- **猫咪城镇**：居民最密集、关系最交错的长期生活中心。

地点决定基础订单类型；NPC 关系、此前投递结果和岛屿变化会让订单内容继续演变。例如灯塔恢复运作会影响沙滩夜间活动，森林队带回新植物会让城镇花店和船上温室出现新订单。

## 成长、声望与家

玩家同时积累全海域声望与地点声望。

- 全海域声望影响航线、合同、稀有订单和邮船经营机会。
- 地点声望来自稳定投递、帮助居民和完成特殊订单；它会解锁当地设施、关系内容，以及属于玩家的住处。

玩家后期可在一个或多个地点获得房屋，例如灯塔旁守望屋、沙滩小屋、森林营地小屋或城镇旧宅。船偏向功能扩建：仓储、冷链、温室、背包和工作台；房屋偏向装修、纪念品展示与生活表达。

## 邮政日志

船上保留爷爷的旧邮政日志。玩家完成特殊订单、发现航线变化或见证居民关系推进后，会加入自己的记录。日志既用于回顾人物、地点和任务，也让爷爷留下的过去与玩家正在建立的新生活并列存在。

## 长期目标

```text
维护并扩展邮船
→ 重新建立岛屿之间的联系
→ 获得各地点的信任与住处
→ 装修邮船与房屋
→ 让爷爷留下的遗物重新成为玩家自己的生活与归属
```
