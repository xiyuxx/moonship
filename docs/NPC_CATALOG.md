# NPC 设定表（第一版）

本文档记录可进入游戏的角色基础设定。首批角色全部生活在「猫咪城镇」，用于支撑新手教学、日常订单与轻量剧情线；他们的故事应当从快递与交谈中逐渐显露，而非一次说明完毕。

## 使用原则

- 日常订单占绝大多数；剧情订单应稀少、短小，且不强迫玩家立刻接下。
- 每位 NPC 都应有：稳定的日常寄件理由、一个可观察的性格特征，以及一条能逐步发现的私事。
- 角色不以“任务发布器”的方式说话。订单文本先说明现实需要，再留下少许情绪或关系线索。
- 初期不使用明确魔法设定；所有物品与故事保持温暖、具体、生活化。

## 0. 阿岚——爷爷的老顾客／新手引导

| 项目 | 设定 |
|---|---|
| 身份 | 年长的橘猫，经营港口旁的旧唱片与留声机修理铺 |
| 性格 | 温和、讲究、记性极好；说话慢，但不故弄玄虚 |
| 与爷爷 | 多年好友。曾常托爷爷寄送修好的留声机零件，也会替爷爷留一盏港口的灯 |
| 初次登场 | 玩家继承邮船后的第一个白天来到接待厅。得知爷爷去世后安静许久，随后提出寄一只旧留声机的发条盒，借此教玩家收件、打包、入库与夜间投递 |
| 日常订单 | 发条盒、唱针、修复用软布、旧唱片、木制喇叭罩 |
| 剧情线索 | 他保留着爷爷年轻时寄来的第一张明信片；后期会交给玩家，但不是谜题的开端，只是关于“这艘船曾经去过哪里”的一段回忆 |
| 说话示例 | “你爷爷总说，信要送到人手里，才算真正离开了船。” |

## 1. 露娜——图书管理员

| 项目 | 设定 |
|---|---|
| 身份 | 灰白长毛猫，城镇图书室管理员 |
| 性格 | 克制、细心、轻微怕生；对书的状态有近乎固执的保护欲 |
| 日常订单 | 旧书修补胶、借阅登记册、花店压花书签、隔壁岛寄来的期刊 |
| 包裹偏好 | 怕潮的纸品，多需常温干燥保存；少量易碎书挡需木箱 |
| 剧情线索 | 图书室一直留着一张空桌。玩家后来才知道，那是她为一位长期在外旅行、已很久没回信的朋友保留的 |
| 说话示例 | “书页不怕旧，怕的是被人忘记翻开。” |

## 2. 诺亚——裁缝

| 项目 | 设定 |
|---|---|
| 身份 | 黑猫，经营小小的裁缝铺，擅长修补衣物与制作旅行用品 |
| 性格 | 直率、手快、嘴上不饶人；其实很愿意照顾镇上的人 |
| 日常订单 | 布料卷、纽扣盒、线轴、雨披、定制布袋、缝纫机零件 |
| 包裹偏好 | 布料要防潮；精细纽扣与针具需防震或木箱 |
| 剧情线索 | 他总给一位匿名客户寄去尺寸很小的旅行斗篷。后期揭示那是给离家加入森林探险队的妹妹准备的 |
| 说话示例 | “衣服破了可以补。人走远了……至少也该带件合身的。” |

## 3. 贝洛——钟楼管理员

| 项目 | 设定 |
|---|---|
| 身份 | 虎斑猫，负责城镇钟楼与港口报时钟的维护 |
| 性格 | 严谨、略显古板、对时间异常敏感；被夸奖时会假装没听见 |
| 日常订单 | 齿轮盒、黄铜螺丝、机芯油、钟摆零件、防水表带 |
| 包裹偏好 | 金属零件多为易碎精密件，要木箱；机芯油须避热 |
| 剧情线索 | 钟楼每晚会比标准时间慢一分钟。贝洛知道原因，却一直没修，因为那一分钟是他与已故伴侣过去约定的散步时间 |
| 说话示例 | “准时不是急着赶路，是让等你的人知道你会来。” |

## 4. 悠悠——小剧团领班

| 项目 | 设定 |
|---|---|
| 身份 | 三花猫，镇上巡回小剧团的领班兼道具管理员 |
| 性格 | 热情、爱夸张、擅长调动气氛；独处时反而有点安静 |
| 日常订单 | 演出布景、戏服羽毛、颜料、节目单、纸花、灯泡 |
| 包裹偏好 | 颜料怕冻，纸花怕潮，玻璃灯泡易碎 |
| 剧情线索 | 剧团的老戏《月光邮船》缺失最后一页。她一直在向各处打听，也许爷爷曾经看过完整版本 |
| 说话示例 | “台下只有一个观众，也要把最后一句话送到他心里。” |

## 5. 阿棉——港口医师

| 项目 | 设定 |
|---|---|
| 身份 | 奶牛猫，港口诊所的医师 |
| 性格 | 冷静、利落、有一点工作狂；会把关心藏进非常具体的叮嘱里 |
| 日常订单 | 冷藏药剂、纱布、草药包、维生素补剂、保温瓶、海盐喷雾 |
| 包裹偏好 | 药剂需冷藏或保温；部分草药需要温室保存 |
| 剧情线索 | 她定期为北礁灯塔的一位老人准备药品，却从不主动谈起对方。后期可作为灯塔地点与玩家建立关联的自然起点 |
| 说话示例 | “这盒药别在甲板上晒着。海风不管它是不是急件。” |

## 6. 芸姐——茶馆主人

| 项目 | 设定 |
|---|---|
| 身份 | 棕色狸花猫，港口茶馆「潮汐壶」的主人 |
| 性格 | 圆融、善于倾听、爱记住客人的小习惯；偶尔会笑着绕开自己的问题 |
| 日常订单 | 上釉茶壶、茶叶罐、点心模具、季节果酱、手写菜单 |
| 包裹偏好 | 陶器易碎，果酱怕热，茶叶怕潮 |
| 剧情线索 | 她替许多人代收“暂时不想面对”的信；玩家可选择只完成投递职责，而非窥探内容。后期她会教玩家尊重邮差与秘密的边界 |
| 说话示例 | “寄信的人未必想要答案。有时只想让那句话有个去处。” |

## 7. 米粒——花店学徒

| 项目 | 设定 |
|---|---|
| 身份 | 年轻的橘白猫，花店「枝叶间」的学徒 |
| 性格 | 开朗、毛手毛脚、容易被新鲜事吸引；对植物异常有耐心 |
| 日常订单 | 鲜花束、温室幼苗、花盆、压花、花种、礼物卡片 |
| 包裹偏好 | 鲜花、幼苗与种子需温室；花盆易碎 |
| 剧情线索 | 她在练习为每位顾客搭配“合适的花”，却始终不敢给自己已搬去沙滩景点工作的好友寄第一束告别花 |
| 说话示例 | “花不是一定要等节日才送呀。今天的风好，也可以。” |

## 8. 洛洛——港口跑腿员

| 项目 | 设定 |
|---|---|
| 身份 | 身形小巧的狸花猫，替店铺与住户跑腿，熟悉城镇每一条小路 |
| 性格 | 机灵、健谈、热心过头；喜欢收集别人不要的邮票边角 |
| 日常订单 | 路线图、便当盒、修理零件、雨伞、各店的小批量补货 |
| 包裹偏好 | 订单多为普通件，适合让玩家熟悉基础流程 |
| 剧情线索 | 他总知道镇上发生什么，但会逐步学会：知道不等于应该替别人说出口 |
| 说话示例 | “我可以带路！不过有些门，还是得你自己敲。” |

## 首批关系网

```text
阿岚 ── 老友 ── 爷爷（已故）
  │
  └── 引导玩家认识城镇与邮船

阿棉 ── 定期寄药 ── 北礁灯塔老人
诺亚 ── 寄旅行斗篷 ── 森林探险队成员（妹妹）
米粒 ── 想寄花 ── 沙滩景点工作的朋友
悠悠 ── 寻找戏本末页 ── 爷爷旧航线／邮政日志
```

## 后续地点的 NPC 配额建议

每个新地点先做 4 至 6 名核心 NPC 即可：一名地点引导者、两至三名提供稳定日常订单的居民、一名带有跨地点联系的角色，以及一名可选的轻量剧情角色。这样能让地点先“像有人住”，再逐步增加故事密度。
